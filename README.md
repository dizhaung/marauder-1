# Artifacts
To use Marauder's artifacts in your project, below is the Maven central signature for artifacts.
```xml
<dependency>
    <groupId>com.srotya.marauder</groupId>
    <artifactId>[name of the project]</artifactId>
    <version>0.0.1</version>
</dependency>
```
